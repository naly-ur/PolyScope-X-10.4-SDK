This folder contains the PolyScope X SDK and simulator.

.devcontainer/     - Contains descriptor for a devcontainer.
sdk-polyscopex/    - Enables developers to create URCap contributions for PolyScope X.
ursim-polyscopex/  - For testing URCap contributions without the use of a real robot.

The SDK and simulator can be installed and used independently, see installation guides in each folder.
